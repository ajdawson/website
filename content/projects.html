Open Source Projects

I've written and contributed to a number of open source projects, most of which are data analysis or scientific computing projects.

Highlights

iris and cartopy

I am a core developer on both these projects. The iris package is a powerful and easy to use Python library for analysing and visualising meteorological and oceanographic data sets. Cartopy is a Python library for cartographic transforms and plotting. Both are community-driven projects, led by the UK Met Office.

eofs

eofs is a Python package for EOF analysis of spatial-temporal data sets. I designed it to be suitable for use on large data sets typical of modern climate model output. It is able to handle missing values transparently, and supports fields with metadata. The package is also bundled with the UV-CDAT distribution.

windspharm

windspharm is a Python package I created for doing computations on global vector fields in spherical geometry. It provides a high level interface for computations involving spherical harmonic transformations, and supports fields with metadata. The package is also bundled with the UV-CDAT distribution.

Other Packages
