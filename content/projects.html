Open Source Projects

eofs

eofs is a Python package for EOF analysis of spatial-temporal data sets. It was written to be suitable for use on large data sets typical of modern climate model output. It is able to handle missing values transparently, and supports fields with metadata using either CDAT or Iris. The package is part of the UV-CDAT distribution.

windspharm

windspharm is a Python package I created for doing computations on global vector fields in spherical geometry. It provides a high level interface for computations involving spherical harmonic transformations. The package supports fields with metadata using CDAT or Iris. The package is part of the UV-CDAT distribution.

Iris

I am a core developer for iris, which is a powerful, easy to use, and community-driven Python library for analysing and visualising meteorological and oceanographic data sets.
