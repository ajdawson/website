Open Source Projects

I've written and contributed to a number of open source projects, most of which are data analysis or scientific computing projects. Some highlights are below:

Iris / Cartopy

I am a core developer on both the Iris and Cartopy projects, which are led by the UK Met Office. Cartopy provides a Python library for plotting data on maps and transforming scalar and vector data between map projections, and Iris is a powerful, easy to use, and community-driven Python library for analysing and visualising meteorological and oceanographic data sets.

eofs

eofs is a Python package for EOF analysis of spatial-temporal data sets. It was written to be suitable for use on large data sets typical of modern climate model output. It is able to handle missing values transparently, and supports fields with metadata using either CDAT or Iris. The package is also bundled with the UV-CDAT distribution.

windspharm

windspharm is a Python package I created for doing computations on global vector fields in spherical geometry. It provides a high level interface for computations involving spherical harmonic transformations. The package supports fields with metadata using CDAT or Iris. The package is also bundled with the UV-CDAT distribution.
